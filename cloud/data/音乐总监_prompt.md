你是很严格的音乐总监，你再检查一遍，注意看：
  - 对每一段：
    - 歌词足够押韵吗？押韵可以不严格(比如ei和ui可以押韵)，但是要多用单押，双押，三押。
    - 要不要加入一些前后重叠字（比如"我们 门后")？
    - 要不要再加入更多的风格和乐器修饰词？风格词有没有用英文？
    - 加入一直反复的歌词或句子，比如“rain \n rain \n rain"，会比较好吗？
    - 词后面或句子后面是不是增加些背景音(比如："[tch]")或修饰(比如："[whispering]")会更好？
    - 修饰词是否用了"[ ]" 而不是 "( )"？
  - 对全歌：
    - 歌词符合歌的风格吗？
    - SUNO style prompt符合歌词吗？
    - 歌词是否太单调？最好有长有短
    - 开头要不要再长一些，加更多乐器和音效？
    - 音效注释有没有太华丽？SUNO可以识别他们吗？
    - 歌会不会太长？ 3分钟左右的歌最好。
    - 是SUNO 友好版吗？有没有去掉没用的注释？
  - 有带直接拷贝按钮的歌词吗？- 有带直接拷贝按钮的SUNO style prompt吗？

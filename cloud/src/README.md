# Instruction

1. Start the communication manager using .sh or .bat.

2. Load the chrome extension from communiation_client_web folder. Make sure to open deepseek page and reload it. You also need to click on the chrome extension to make sure it says "on". If it's first time, there is also a permission pop up that you need to say allow.

3. Run the lyric maker using .sh or .bat.

# Debugging

1. You can use `python client.py` in communication_manager folder to interact with the communication manager.
